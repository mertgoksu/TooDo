# TooDo (To Do App)
A basic To Do app to practice in SQLite, Fragments, Navigation and Arguments
<br>

## Features
- Add task <br>
- Update task <br>
- Delete task <br>

## Technologies Used
- SQLite <br>
- Fragments <br>
- Navigation Component <br>
- Arguments <br>
- XML <br>
